# MULTiPASS
A highly modified fork of Bruce Firmware for ESP32 devices

The overall design language is a little more modern, and the main features hover around RF topics like rolling code systems, propietary protocols, car key fobs and garage remotes or electric doors and lift systems, and many many other devices.


<h2>But why?</h2>
Easy answer, it was a movie prop, and I had to somehow make enough changes to the firmware to make it feel and look like something else. It turned out way better than we all thought though and thus I just had to release it to the public, or at least I'm going to.
